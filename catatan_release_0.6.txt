********************************
Catatan Release v0.6
********************************

Sejak 22 Juni 2016, semua website yang memanggil Google API, termasuk Google Maps, harus memakai Google API Key.

Di rilis v0.6, semua pemanggilan Google Maps API di OpenSID membutuhkan Google API Key. Key tersebut perlu diperoleh dari Google. Untuk informasi lebih lanjut, lihat https://developers.google.com/maps/documentation/javascript/get-api-key dan https://www.latecnosfera.com/2016/06/google-maps-api-error-missing-keymap-error-solved.html.

Key yang diperoleh perlu dimasukkan ke dalam file konfigurasi sid.ini di folder desa/config.

Contoh file sid.ini disediakan di folder desa-contoh/config/
