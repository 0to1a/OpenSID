********************************
Catatan Release v0.7.2
********************************

Rilis ini berisi perubahan pada file desa-contoh/css/desa-web.css.
Perubahan ini untuk memperbaiki penampilan artikel pada contoh oprekan penampilan web.

Untuk menerapkan perubahan yang dilakukan, anda dapat:

*   mencopy file desa-web.css dan menyalinnya ke folder desa/css. Lakukan ini hanya apabila anda
    tidak melakukan oprekan lainnya di file desa/css/desa-web.css
*   mempelajari perubahan yang dilakukan dan memilih styles yang diinginkan untuk dimasukkan ke desa/css/desa-web.css.
    Perubahan yang dilakukan dapat dilihat di https://github.com/eddieridwan/OpenSID/commit/7e05ffc6cefbf919229b9509c13fdb4a2bf5d87d
